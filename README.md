## ‚ú® Deskripsi Proyek

Proyek ini adalah sebuah website portofolio yang dirancang untuk menampilkan profil, latar belakang akademik, dan minat saya dalam pengembangan perangkat lunak dan teknologi informasi. Seluruh _styling_ proyek ini menggunakan **CSS native** dengan penerapan praktik modern seperti **variabel CSS** dan tata letak **Flexbox/Grid**.

**Fitur Utama:**

1.  **Hero Section:** Sapaan pembuka yang memperkenalkan diri.
2.  **Tentang Saya:** Informasi detail mengenai latar belakang studi, ketertarikan pada pemrograman, basis data, jaringan komputer, dan rekayasa perangkat lunak.
3.  **Artikel Terbaru:** Section yang memuat artikel-artikel terkait analisis strategis dan teknologi.
4.  **Kontak:** Menyediakan formulir dan detail kontak langsung (Email, Telepon, dan Lokasi Kampus).

## ‚öôÔ∏è Langkah-langkah Instalasi dan Penggunaan

Proyek ini adalah aset web statis (HTML/CSS) dan tidak memerlukan _framework_ atau _library_ pihak ketiga yang kompleks.

### Instalasi

Untuk mendapatkan salinan proyek ini dan menjalankannya secara lokal:

1.  **Clone Repositori:**
    Buka terminal atau Git Bash Anda dan _clone_ repositori ini ke komputer Anda:

    ```bash
    git clone [https://github.com/inthemiddleof/tugas-akhir-ppw-2.git](https://github.com/inthemiddleof/tugas-akhir-ppw-2.git)
    cd tugas-akhir-ppw-2
    ```

2.  **Unduh Aset (Opsional):**
    Pastikan Anda memiliki semua folder aset yang diperlukan, terutama folder `assets/img/` yang berisi gambar profil, dan gambar artikel (`swot.jpg`, `porter.jpg`, dll.) agar tampilan tidak rusak.

### Penggunaan (Menjalankan Secara Lokal)

1.  **Buka File:**
    Cukup buka file `index.html` yang ada di _root_ folder proyek dengan _browser_ pilihan Anda (Chrome, Firefox, Edge, dll.).

    _Anda bisa klik kanan pada `index.html` dan pilih "Open with..." lalu pilih browser Anda._

2.  **Modifikasi:**
    Anda dapat memodifikasi _styling_ melalui tag `<style>` di masing-masing file html atau memisahkan CSS ke file eksternal untuk kemudahan pengembangan lebih lanjut.
